## 3º) GIT
```
git config --global user.name "John Doe"
git config --global user.email johndoe@example.com
```
Añadir alias en `.gitconfig`

```
[user]
        name = Conchi Martín
        email = konchimartin@gmail.com
[alias]
        co = checkout
        br = branch
        ci = commit
        st = status
```
